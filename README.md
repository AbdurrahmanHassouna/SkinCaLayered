# SkinCA API (Skin Cancer Detection App Backend)

[![API Version](https://img.shields.io/badge/Version-1.0-blue)]()

## Overview

This is the backend API for **SkinCA**, a health application focused on skin cancer awareness, detection, and management. It provides features such as:

- User and doctor registration/authentication
- Disease information management
- Machine learning-based skin analysis (image upload for prediction)
- Doctor directory and nearby search
- Chat messaging between users/doctors
- Medical reports, bookmarks, reminders, and banners
- Profile management and account features

The API is built with ASP.NET Core and exposes a Swagger/OpenAPI specification.

**Live Swagger Documentation:** [https://skinca.runasp.net/swagger](https://skinca.runasp.net/swagger)  
**OpenAPI JSON:** [https://skinca.runasp.net/swagger/v1/swagger.json](https://skinca.runasp.net/swagger/v1/swagger.json)
